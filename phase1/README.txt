To run the program for the first time, please edit the configuration of the Main class to run with an argument of 0.
To run the program subsequent times and ensure data persists, run the Main class with an argument of 1.

To run the program, the opencsv library must be installed.
Navigate to File -> Project Structure -> Modules -> Dependencies and use the + symbol to add the opencsv-5.2.jar file.

The initial admin username is "1" and the password is "blerg".
The number of trades must be reset each week manually from the admin menu.
